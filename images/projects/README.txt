# 📸 Project Screenshots

Place your project screenshots here.

## Recommended Specs:
- **Format**: PNG or JPG
- **Dimensions**: 800x450px (16:9 ratio) or similar
- **File naming**: Use descriptive names with hyphens
  - ✅ Good: `space-shooter-game.png`, `vr-simulation.jpg`
  - ❌ Avoid: `Screenshot 1.png`, `IMG_001.jpg`

## Usage:
After adding images here, update `js/content.js`:

```javascript
{
    title: "Your Project",
    image: "images/projects/your-screenshot.png",  // ← Update this path
    // ...
}
```

## Example filenames:
- `air-assault-gameplay.png`
- `racing-game-screenshot.jpg`
- `vr-experience-cover.png`

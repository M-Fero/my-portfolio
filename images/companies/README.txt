# 🏢 Company Logos

Place your company/employer logos here.

## Recommended Specs:
- **Format**: PNG with transparent background (preferred)
- **Dimensions**: 200x200px or larger (will be auto-resized)
- **File naming**: Use company name with hyphens
  - ✅ Good: `arcon-logo.png`, `vodafone-logo.png`
  - ❌ Avoid: `Logo 1.png`, `company.jpg`

## Usage:
After adding logos here, update `js/content.js`:

```javascript
{
    position: "Your Position",
    company: "Company Name",
    logo: "images/companies/company-logo.png",  // ← Update this path
    // ...
}
```

## Example filenames:
- `unity-technologies.png`
- `arcon-logo.png`
- `vodafone-logo.png`
- `microsoft-logo.png`

# 📄 CV & Documents

Place your CV/Resume PDF files here.

## Recommended:
- **Format**: PDF
- **File naming**: Use your name and year
  - ✅ Good: `Mostafa-Fero-CV-2025.pdf`, `John-Doe-Resume.pdf`
  - ❌ Avoid: `CV.pdf`, `Resume Final Final.pdf`

## Usage:
After adding your CV here, update `js/content.js`:

```javascript
const personalInfo = {
    // ...
    cvFile: "assets/Your-Name-CV-2025.pdf",  // ← Update this path
    // ...
};
```

## Tips:
- Keep file size under 5MB for faster loading
- Update your CV regularly
- You can have multiple versions (e.g., detailed vs. short)
